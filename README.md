## Запуск сервера

```
server [-p port]
```
Порт - целое число в диапазоне 0-65535. По умолчанию используется значение 8080

## Структура для хранения задач

**id** - целое число, задается автоматически при создании задачи, не подлежит изменению.  
**title** - название задачи, должно быть передано в теле запроса при создании и изменении.  
**done** - статус завершения. По умолчанию при создании false.  
**created_at** - время создания, заполняется автоматически при создании задачи, не подлежит изменению.


## Эндпоинты

| Метод | Эндпоинт | Описание | Комментарий |
| ----- | -------- | -------- | ----------- |
| GET | /tasks | получение списка задач | |
| POST | /tasks | создание новой задачи | название и статус задачи надо передавать в теле запроса в JSON структуре|
| GET | /tasks/{id} | просмотр задачи | |
| PUT | /tasks/{id} | изменение задачи| название и статус задачи надо передавать в теле запроса в JSON структуре |
| DELETE | /tasks/{id} | удаление задачи | |

## Пример использования

#### Получение списка задач

```bash
>curl http://localhost:8080/tasks

[{"id":1,"title":"Дефрагментация диска","done":false,"created_at":"2025-11-24T16:24:16+03:00"},{"id":2,"title":"Сборка мусора","done":false,"created_at":"2025-11-24T16:24:16+03:00"},{"id":3,"title":"Оптимизация запросов","done":false,"created_at":"2025-11-24T16:24:16+03:00"}]
```

#### Cоздание новой задачи
```bash
>curl -X POST -H "Content-Type: application/json" --data "{\"title\": \"Создание индексов\", \"done\": false}" http://localhost:8080/tasks

{"id":4,"title":"Создание индексов","done":false,"created_at":"2025-11-30T20:42:07+03:00"}
```

#### Просмотр задачи
```bash
>curl http://localhost:8080/tasks/3

{"id":3,"title":"Оптимизация запросов","done":false,"created_at":"2025-11-24T16:24:16+03:00"}
```

#### Изменение задачи
```bash
>curl -X PUT -H "Content-Type: application/json" --data "{\"done\": true}" http://localhost:8080/tasks/4

{"status":"Task succesfully updated"}
```

#### Удаление задачи
```bash
>curl -X DELETE http://localhost:8080/tasks/4

# тело ответа пустое, в заголовке передается статус NoContent - 204
```
Повторный запрос к уже удаленной задаче приведет к ответу со статусом 404
```bash
>curl -X DELETE http://localhost:8080/tasks/4

{"error":"Wrong id"}
```
